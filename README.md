# Virtual Pet Maker

A small object-oriented Python project: create a virtual pet, then feed, play
with, rest, and check on it from a simple command-line menu. Each action shifts
the pet's hunger, happiness, and energy stats over time.

Started as a hands-on project for practicing classes and state management — and
it's the direct ancestor of **Hatch**, my later browser-based virtual pet.

## Run it
```bash
python VPM.py
```
Enter a name, then choose actions from the menu (feed / play / sleep / status).

## Concepts
Python classes · encapsulated state · input loops · basic game logic
